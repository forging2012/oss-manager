# A Simple OSS Manager

A rails app demo for (the powerful) [OSS Ruby SDK][oss-ruby-sdk].

## Getting started

Run in your terminal:

    git clone git@gitlab.alibaba-inc.com:tianlong.wtl/oss-manager.git
    cd oss-manager
    bundle install
    ./bin/rails server
    # visit http://localhost:3000/buckets/
